# RWD

Prosta demonstracyjna responsywna strona internetowa w html, css i js.

## Opis
Strona prezentuje podstawowe zasady Responsive Web Design (RWD).
Zawiera nagłówek z menu, sekcję z trzema elementami oraz stopkę.

## Funkcjonalności
- Responsywny układ (telefony, tablety, srednie ekrany, duze ekrany)
- Menu poziome na dużych ekranach
- Menu hamburger na małych ekranach
- Rozwijane menu po kliknięciu
- Galeria obrazów (rozny uklad w zaleznosci od rozmiaru okna)

## Zakresy responsywności
- <576px - telefony
- 576–768px - tablety
- 768–992px - średnie ekrany
- >992px - duże ekrany

## Technologie
- HTML
- CSS
- JavaScript
- Google Fonts (Roboto)

## Animacje
- Hover na przyciskach (zmiana koloru)
- Hover na kartach (uniesienie)
- Hover na obrazach (powiększenie)

## Pliki
- `index.html` – struktura strony
- `style.css` – stylowanie
- `README.md` – opis projektu

## Testowanie
Stronę można testować zmieniając szerokość okna przeglądarki.
Układ dostosowuje się automatycznie do rozmiaru ekranu.
